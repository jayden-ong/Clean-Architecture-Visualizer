## Verifying Clean Architecture in Your Repository

This is a tool to help develop your CSC207 Project! With this tool you can initiate your project, create new use cases, and verify your project's adherence to Clean Architecture.

### Installation

You can install the `cave` package from Github with:

```bash
npm install -g "git+https://github.com/CA-Visualizer-for-Education/Clean-Architecture-Visualizer.git#main&path:clean-architecture-visualizer"
```
